# Ghaziexam - Sistem Ujian Online (CBT)

Aplikasi ujian online berbasis web sederhana menggunakan PHP Native dan MySQL.

## Fitur
- Login Siswa & Admin
- Timer Mundur Realtime
- Soal Pilihan Ganda & Esai
- Support Gambar & Audio pada Soal
- Rekap Nilai Otomatis
- Anti-Cheat Sederhana (Fullscreen Warning)

## Cara Instalasi
1. Clone repositori ini.
2. Import database `ujian_db.sql` ke phpMyAdmin.
3. Ubah nama file `koneksi.php.example` menjadi `koneksi.php`, lalu sesuaikan user/pass database lokal Anda.
4. Buka di browser (localhost).
